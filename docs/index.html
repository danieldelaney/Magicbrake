What does it do?

It takes any weird video file and makes it normal—a small, fast, and widely compatible MP4.

How does it work?

Drag the app into your Applications, open it, drag video files onto it, and click the button.

Is it free?

Yes!

Is it open source?

Yes, the source code is freely available.

Is this just Handbrake?

Yes! It’s a simple user interface for the handbrakecli utility that anyone can use.
